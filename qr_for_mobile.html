QR para móvil

Genera un código QR para una URL pública que pueda abrirse desde cualquier red externa.

Si ya tienes una URL de túnel público (por ejemplo de npm run public-qr), pégala aquí y aparecerá el QR. También puedes usar esta página con `?url=` en la URL.

Ejemplo: https://xxxxx.loca.lt/fr/PREGUNTA1.html
